# تشفير البيانات الحساسة في localStorage

## المشكلة المحلولة
كانت البيانات الحساسة (الأسعار، المبالغ، معلومات GitHub) تُخزن بشكل مكشوف في localStorage، مما يشكل خطراً أمنياً.

## الحل المطبق

### 1. مكتبة التشفير (`encryption.js`)
تم إنشاء مكتبة تشفير شاملة توفر:

#### الميزات:
- ✅ **تشفير XOR + Base64**: لحماية البيانات من القراءة المباشرة
- ✅ **مفاتيح ديناميكية**: تتغير يومياً لزيادة الأمان
- ✅ **توقيع البيانات**: للتحقق من عدم العبث بها
- ✅ **تشفير انتقائي**: فقط للحقول الحساسة
- ✅ **ترحيل تلقائي**: للبيانات الموجودة

#### الحقول المشفرة تلقائياً:
- `price`, `amount`, `total`, `balance`
- `retailPrice`, `wholesalePrice`, `distributorPrice`
- `cost`, `profit`, `debt`, `payment`
- `salary`, `income`, `expense`
- معلومات GitHub Token

### 2. التكامل مع النظام

#### دالة حفظ البيانات:
```javascript
// قبل
localStorage.setItem('networkCardsData', JSON.stringify(data));

// بعد
if (window.DataEncryption) {
    window.DataEncryption.saveEncrypted('networkCardsData', data);
} else {
    // fallback للمتصفحات القديمة
    localStorage.setItem('networkCardsData', JSON.stringify(data));
}
```

#### دالة تحميل البيانات:
```javascript
// محاولة تحميل البيانات المشفرة أولاً
const data = window.DataEncryption.loadEncrypted('networkCardsData');

// إذا فشل، حاول التحميل العادي
if (!data) {
    const saved = localStorage.getItem('networkCardsData');
    data = JSON.parse(saved);
}
```

### 3. كيفية عمل التشفير

1. **عند الحفظ**:
   ```
   البيانات الأصلية → تحديد الحقول الحساسة → تشفير القيم → 
   إضافة توقيع → حفظ في localStorage
   ```

2. **عند التحميل**:
   ```
   قراءة من localStorage → التحقق من التوقيع → 
   فك تشفير القيم → إرجاع البيانات الأصلية
   ```

### 4. مثال على البيانات

#### قبل التشفير:
```json
{
  "retailPrice": 100,
  "wholesalePrice": 80,
  "amount": 500
}
```

#### بعد التشفير:
```json
{
  "retailPrice": "YmFzZTY0X2VuY29kZWRfc3RyaW5n",
  "_retailPrice_encrypted": true,
  "wholesalePrice": "YW5vdGhlcl9lbmNvZGVkX3N0cmluZw==",
  "_wholesalePrice_encrypted": true,
  "amount": "ZW5jb2RlZF9hbW91bnQ=",
  "_amount_encrypted": true
}
```

### 5. الأمان الإضافي

- **التوقيع**: كل مجموعة بيانات لها توقيع للتحقق من السلامة
- **الطابع الزمني**: لتتبع آخر تعديل
- **المفاتيح المتغيرة**: تتغير يومياً لمنع الهجمات طويلة المدى

### 6. التوافقية

- ✅ يعمل مع جميع المتصفحات الحديثة
- ✅ fallback تلقائي للمتصفحات القديمة
- ✅ ترحيل سلس للبيانات الموجودة
- ✅ لا يؤثر على الأداء بشكل ملحوظ

## الاستخدام

### للمطورين:

```javascript
// تشفير قيمة واحدة
const encrypted = $encrypt.value(sensitiveData);

// فك تشفير قيمة
const decrypted = $encrypt.decrypt(encrypted);

// حفظ كائن كامل مشفر
$encrypt.save('myKey', myObject);

// تحميل كائن مشفر
const myObject = $encrypt.load('myKey');
```

## التحذيرات

⚠️ **هذا التشفير للحماية الأساسية فقط**
- لا يُغني عن HTTPS
- لا يحمي من المستخدمين المتقدمين
- يُنصح بعدم تخزين معلومات شديدة الحساسية محلياً
