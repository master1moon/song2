# زر "فتح صفحة التقرير" - شرح مفصل

## 📊 ما هو التقرير؟

عند الضغط على زر "فتح صفحة التقرير" الأزرق، يتم فتح نافذة جديدة بصفحة HTML مصممة خصيصاً للطباعة تحتوي على تقرير مفصل.

## 🎯 محتويات التقرير

### 1. **رأس التقرير:**
- عنوان التقرير (مثل: "كشف حساب: محل العربي")
- الفترة الزمنية المحددة
- تاريخ إنشاء التقرير

### 2. **الملخص المالي:**
- إجمالي المبيعات
- إجمالي التسديدات
- المبلغ المتبقي (دائن/مدين)

### 3. **جدول المبيعات المفصل:**
- التاريخ
- التفاصيل (سبب البيع)
- اسم الباقة
- الكمية أو المبلغ
- الإجمالي

### 4. **جدول التسديدات المفصل:**
- التاريخ
- المبلغ
- الملاحظات

## 🎨 مميزات التقرير

### 1. **تصميم احترافي:**
- خط عربي واضح (Amiri)
- جداول منظمة ومرتبة
- ألوان مريحة للعين

### 2. **قابل للطباعة:**
- زر "حفظ التقرير كـ PDF"
- تنسيق محسن للطباعة
- إخفاء الأزرار عند الطباعة

### 3. **مرونة في الفترة:**
- يمكن تحديد فترة زمنية محددة
- أو عرض كل البيانات

### 4. **معلومات شاملة:**
- كل المبيعات بالتفصيل
- كل التسديدات بالتفصيل
- حساب الرصيد تلقائياً

## 💡 الاستخدامات

1. **للمحاسبة:** طباعة كشف حساب شهري
2. **للمراجعة:** مراجعة حساب محل معين
3. **للأرشفة:** حفظ نسخة PDF للسجلات
4. **للعملاء:** إرسال كشف حساب للمحل

## 🖨️ كيفية الحفظ كـ PDF

1. اضغط على زر "فتح صفحة التقرير"
2. في الصفحة الجديدة، اضغط "حفظ التقرير كـ PDF"
3. أو استخدم Ctrl+P (أو Cmd+P على Mac)
4. اختر "Save as PDF" من قائمة الطابعات
5. احفظ الملف

## 📝 ملاحظات
- التقرير يُعرض بصيغة HTML وليس PDF مباشرة
- يمكن للمستخدم طباعته أو حفظه كـ PDF
- البيانات محدثة لحظة إنشاء التقرير
- يدعم الفترات الزمنية المخصصة
