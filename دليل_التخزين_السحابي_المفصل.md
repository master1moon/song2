# دليل التخزين السحابي المفصل - خطوة بخطوة

## 🚀 حلول Google Drive

### الحل الأول: استخدام Google Colab (الأسهل والأسرع)

#### الخطوات بالتفصيل:

1. **افتح Google Colab**
   - اذهب إلى: https://colab.research.google.com
   - سجل دخول بحساب Google الخاص بك

2. **أنشئ دفتر ملاحظات جديد**
   - اضغط على "New notebook" أو "دفتر ملاحظات جديد"

3. **في التطبيق، قم بإنشاء نسخة احتياطية**
   - اذهب للإعدادات → النسخ الاحتياطي
   - اختر "Google Drive" من مكان الحفظ
   - اضغط "إنشاء نسخة احتياطية الآن"
   - ستظهر نافذة بها كود Python

4. **انسخ الكود من النافذة**
   - اضغط على زر "نسخ الكود"
   - الكود سيكون مثل هذا:
   ```python
   # كود Python لحفظ النسخة الاحتياطية في Google Drive
   from google.colab import drive
   import json
   from datetime import datetime
   
   # ربط Google Drive
   drive.mount('/content/drive')
   
   # البيانات (ستكون بياناتك الفعلية هنا)
   backup_data = '''{"version":"2.0","timestamp":"2024-01-15..."}'''
   
   # حفظ الملف
   filename = f"backup_{datetime.now().strftime('%Y%m%d_%H%M%S')}.json"
   filepath = f"/content/drive/MyDrive/{filename}"
   
   with open(filepath, 'w', encoding='utf-8') as f:
       f.write(backup_data)
   
   print(f"✅ تم حفظ النسخة في: {filename}")
   ```

5. **الصق الكود في Colab**
   - الصق الكود في خلية Colab
   - اضغط على زر التشغيل (▶️) أو Ctrl+Enter

6. **اسمح بالوصول لـ Drive**
   - ستظهر نافذة تطلب السماح
   - اضغط "Connect to Google Drive"
   - اختر حسابك
   - اسمح بالوصول

7. **انتظر اكتمال الحفظ**
   - سترى رسالة: "✅ تم حفظ النسخة في: backup_20240115_143045.json"
   - افتح Google Drive الخاص بك
   - ستجد الملف في المجلد الرئيسي

### الحل الثاني: استخدام البريد الإلكتروني

#### الخطوات:

1. **أنشئ نسخة احتياطية**
   - نفس الخطوات السابقة
   - ستظهر خيارات مختلفة

2. **اضغط على "إرسال بالبريد"**

   - الموضوع والمحتوى جاهزان

3. **أرسل لنفسك**
   - غيّر البريد إلى: your-email@gmail.com

   - افتح Gmail
   - افتح الرسالة
   - انسخ محتوى النسخة الاحتياطية
   - افتح Google Drive
   - أنشئ مستند جديد (New → Google Docs)
   - الصق المحتوى
   - غيّر الاسم إلى backup_التاريخ.json

## 📦 حلول Dropbox

### الحل الأول: Send to Dropbox (مجاني وسهل)

#### الخطوات:

1. **قم بالتسجيل في Send to Dropbox**
   - اذهب إلى: https://sendtodropbox.com
   - اضغط "Sign up with Dropbox"
   - سجل دخول بحساب Dropbox

2. **احصل على بريدك الخاص**
   - بعد التسجيل، ستحصل على بريد مثل:
   - `yourname.a1b2c3d4@sendtodropbox.com`
   - احفظ هذا البريد

3. **في التطبيق**
   - أنشئ نسخة احتياطية
   - اختر "محلي" أولاً لتحميل الملف
   - الملف سيُحفظ في مجلد التنزيلات

4. **أرسل الملف بالبريد**
   - افتح بريدك الإلكتروني
   - أنشئ رسالة جديدة
   - المرسل إليه: بريد sendtodropbox الخاص بك
   - أرفق ملف النسخة الاحتياطية
   - أرسل

5. **تحقق من Dropbox**
   - افتح Dropbox
   - ستجد مجلد "Apps/Send to Dropbox"
   - الملف موجود هناك!

### الحل الثاني: IFTTT (أتمتة كاملة)

#### الخطوات:

1. **أنشئ حساب IFTTT**
   - اذهب إلى: https://ifttt.com
   - سجل مجاناً
   - اربط حساب Gmail و Dropbox

2. **أنشئ Applet جديد**
   - اضغط "Create"
   - If This: اختر "Gmail"
   - اختر "New email in inbox from search"
   - في البحث اكتب: `subject:"نسخة احتياطية"`

3. **Then That**
   - اختر "Dropbox"
   - اختر "Create a text file"
   - File name: `backup_{{ReceivedAt}}.json`
   - Content: `{{BodyPlain}}`
   - Dropbox folder: `/Backups`

4. **في التطبيق**
   - استخدم خيار "إرسال بالبريد"
   - غيّر الموضوع إلى: "نسخة احتياطية"
   - أرسل لنفسك
   - IFTTT سيحفظها تلقائياً في Dropbox!

## ⚡ الحل الأقوى: Zapier (للمحترفين)

### إعداد Zapier مرة واحدة:

1. **أنشئ حساب Zapier**
   - اذهب إلى: https://zapier.com
   - الخطة المجانية تكفي (100 مهمة/شهر)

2. **أنشئ Zap جديد**
   - Trigger: Webhooks by Zapier
   - Event: Catch Hook
   - ستحصل على رابط مثل:
   ```
   https://hooks.zapier.com/hooks/catch/123456/abcdef/
   ```

3. **Action**
   - اختر Google Drive أو Dropbox
   - Event: Create File from Text
   - ربط الحساب
   - File Name: `backup_{{timestamp}}.json`
   - File Content: `{{raw_body}}`

4. **في التطبيق (مرة واحدة فقط)**
   - احفظ رابط Webhook في مكان آمن
   - يمكنك إضافته في إعدادات التطبيق

5. **للاستخدام**
   - كل ما عليك هو إرسال البيانات للرابط
   - سيحفظ تلقائياً في السحابة!

## 🌐 حل WebDAV (يعمل مع خدمات كثيرة)

### خدمات تدعم WebDAV:
- **Box.com** (مجاني 10GB)
- **Yandex.Disk** (مجاني 10GB)
- **pCloud** (مجاني 10GB)
- **Nextcloud** (استضافة ذاتية)

### مثال مع Box.com:

1. **أنشئ حساب Box**
   - اذهب إلى: https://box.com
   - سجل مجاناً

2. **فعّل WebDAV**
   - الإعدادات → الأمان → إنشاء كلمة مرور تطبيق
   - احفظ كلمة المرور

3. **استخدم تطبيق WebDAV**
   - حمّل تطبيق مثل "WebDAV Navigator" للجوال
   - أو استخدم "Cyberduck" للكمبيوتر
   - الخادم: `https://dav.box.com/dav`
   - المستخدم: بريدك الإلكتروني
   - كلمة المرور: كلمة مرور التطبيق

4. **ارفع النسخة**
   - احفظ النسخة محلياً من التطبيق
   - ارفعها عبر WebDAV
