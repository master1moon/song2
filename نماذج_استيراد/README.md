# نماذج ملفات الاستيراد

هذا المجلد يحتوي على نماذج لملفات JSON الصحيحة للاستيراد في النظام.

## أنواع الملفات المدعومة:

### 1. استيراد جميع البيانات
- استخدم `all_data_example.json` لاستيراد جميع أنواع البيانات دفعة واحدة

### 2. استيراد الباقات فقط
- استخدم `packages_example.json` لاستيراد الباقات والأسعار

### 3. استيراد المخزون فقط
- استخدم `inventory_example.json` لاستيراد كميات الكروت

### 4. استيراد المحلات فقط
- استخدم `stores_example.json` لاستيراد بيانات المحلات

### 5. استيراد المصروفات فقط
- استخدم `expenses_example.json` لاستيراد المصروفات

### 6. استيراد المبيعات فقط
- استخدم `sales_example.json` لاستيراد عمليات البيع

### 7. استيراد المدفوعات فقط
- استخدم `payments_example.json` لاستيراد المدفوعات

## كيفية الاستخدام:
1. اختر النموذج المناسب
2. عدّل البيانات حسب حاجتك
3. احفظ الملف بصيغة JSON
4. استخدم خيار "استيراد البيانات" في النظام
5. اختر نوع البيانات المناسب
6. اختر الملف وانقر "استيراد"
