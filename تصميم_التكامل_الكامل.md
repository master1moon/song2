## 🔧 التطبيق التقني:

### 1. **كائن الفلترة الموحد:**
```javascript
const storeFilter = {
  // النوع: دورة مالية، زمني، مخصص
  type: 'cycle', // 'cycle' | 'time' | 'custom'
  
  // معرف الفلتر
  id: 'current_cycle', // 'current_cycle' | 'last_7_days' | 'custom_range' etc
  
  // البيانات
  data: {
    startDate: '2025-07-15',
    endDate: null, // null = حتى الآن
    cycleNumber: 1, // رقم الدورة إن كانت دورة مالية
    includeTypes: ['sales', 'payments'], // أنواع العمليات
  },
  
  // الوصف للعرض
  description: 'الدورة المالية الحالية',
  subtitle: 'من 15 يوليو - حتى الآن'
};
```

### 2. **دالة الفلترة المركزية:**
```javascript
function applyStoreFilter(allData, filter) {
  let filtered = {
    sales: [],
    payments: []
  };
  
  // تطبيق الفلترة حسب النوع
  switch(filter.type) {
    case 'cycle':
      filtered = filterByCycle(allData, filter.data.cycleNumber);
      break;
    case 'time':
      filtered = filterByDateRange(allData, filter.data.startDate, filter.data.endDate);
      break;
    case 'custom':
      filtered = filterCustom(allData, filter.data);
      break;
  }
  
  // تطبيق فلاتر النوع (مبيعات/تسديدات)
  if (!filter.data.includeTypes.includes('sales')) {
    filtered.sales = [];
  }
  if (!filter.data.includeTypes.includes('payments')) {
    filtered.payments = [];
  }
  
  return filtered;
}
```

```javascript
async function exportStoreData(storeId, format) {
  const fromDate = document.getElementById('storeFromDate').value;
  const toDate = document.getElementById('storeToDate').value;
  // ...
}

// بعد: التصدير يستخدم الفلترة النشطة
async function exportStoreData(storeId, format) {
  // استخدام الفلترة الحالية
  const currentFilter = getActiveStoreFilter();
  const filteredData = applyStoreFilter(getAllStoreData(storeId), currentFilter);
  
  // إضافة معلومات الفلترة للتقرير
  const reportMeta = {
    storeName: getStoreName(storeId),
    filterType: currentFilter.description,
    period: currentFilter.subtitle,
    generatedAt: new Date()
  };
  
  // التصدير مع البيانات المفلترة
  switch(format) {
    case 'excel':
      exportToExcel(filteredData, reportMeta);
      break;
    case 'pdf':
      exportToPDF(filteredData, reportMeta);
      break;
    case 'statement':
      exportAccountStatement(filteredData, reportMeta);
      break;
  }
}
```

## ✅ الفوائد:

1. **تجربة موحدة**: ما تراه هو ما تحصل عليه عند التصدير
2. **سهولة الاستخدام**: فلترة واحدة بدلاً من تكرار التواريخ
3. **دقة أكبر**: لا يمكن تصدير بيانات مختلفة عن المعروضة
4. **مرونة**: سهولة إضافة فلاتر جديدة في المستقبل

## 🚀 خطوات التنفيذ:

1. إنشاء مكون الفلترة المتقدم
2. تحديث دوال العرض لاستخدام الفلترة
3. ربط التصدير بالفلترة النشطة
4. إضافة مؤشرات بصرية للحالة الحالية
5. اختبار التكامل الكامل
